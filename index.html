권태수

회원 관련(로그인, 회원가입, 마이페이지) / DB 설계 및 구축

선우세웅

상품 상세 및 결제 기능, 장바구니 기능 구현 / DB 설계 및 구축

장국한

메인 페이지 구현, 아이템 검색 기능 구현 / DB 설계 및 구축

김학성

관리자 페이지 기능 구현 / DB 설계 및 구축

로그인 페이지

로그인 페이지는 로그인,회원가입,아이디/비밀번호 찾기로 구성되어 있습니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)

메인 페이지

메인페이지는 상품나열 및 왼쪽에 화면에 맞춰 움직이는 사이드바로 구성되어 있습니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)

관리자 페이지

관리자 페이지는 상품 등록/관리,관리자 등록/관리, 주문현황으로 구성되어 있습니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)

상품 페이지


상품 페이지는 상품이 한눈에 보이게 나열되어 있으며 카테고리와 2차카테고리를 통해 상품을 볼 수 있습니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)

상품 상세페이지

상품 상세페이지는 메인페이지 혹은 상품페이지에서 상품이미지 클릭 시 진입가능하며
상품수량,판매가,장바구니담기,상품상세설명,상품리뷰로 구성되어 있습니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)

장바구니 페이지

장바구니 페이지는 장바구니에 담은 상품을 보고 주문이 가능한 페이지입니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)

결제 페이지

결제 페이지는 장바구니에 등록한 상품,로그인한 유저의 이름과 주소, 그리고 최종결제를 이용할 수 있는 페이지입니다
(클릭 시 사진이 확대 or 기능이용 사진이 나옵니다)
